"독도는 한국 땅" 이라는 프로젝트에 대해 소개를 시작하겠습니다.  

# 시작하게 된 계기  
독도는 한국 땅 이라는 프로젝트를 시작하게 된 것은 일본이 "다케시마는 일본땅이다." 내용의 웹사이트를 만들어 "독도.com"이라는 주소로 접속 시, 해당 내용이 보이게 해 뒀습니다.  
아직 학생이지만, 독도는 한국 땅 이라는 내용을 강조하고, 전 세계에 알리기 위해 시작 되었습니다.
